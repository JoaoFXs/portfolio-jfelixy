Sobre mim

Sou formado em Ciência da Computação e atualmente trabalho como Desenvolvedor Java na IBM, contribuindo para um projeto da empresa VALE. Tenho experiência em desenvolvimento backend utilizando Java e o ecossistema Spring, incluindo Spring Boot, Spring MVC, Spring Data, Spring Security e Spring Cloud. No frontend, trabalho com React.js, utilizando ferramentas modernas como JavaScript, TypeScript, Tailwind CSS e React Router para construir interfaces de usuário rápidas e responsivas. Também possuo conhecimento em computação em nuvem, com experiência prática em Microsoft Azure e Amazon Web Services (AWS), aplicando as melhores práticas em arquitetura, segurança e escalabilidade.
