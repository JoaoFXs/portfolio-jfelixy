Meus Projetos

CLIMASYNC

ClimaSync é uma plataforma de integração de dados climáticos desenvolvida com Apache Camel. Ela busca informações do clima (temperatura, umidade, previsões) a partir da API OpenMeteo, transforma e valida os dados com XSLT/XSD, e os distribui para sistemas por meio de filas ActiveMQ e armazenamento em MySQL.

IMAGE VAULT

ImageVault é uma aplicação para armazenar, organizar e buscar imagens de forma inteligente, com suporte a filtros por extensão e integração com autenticação de usuários. Ideal para gerenciar bibliotecas visuais com segurança e praticidade.

ADOPTION PET - EM DESENVOLVIMENTO

Sistema de adoção de animais utilizando Spring boot e react.
